# Direction Maze Game

**Description**  
A simple 2D maze game built with C# Windows Forms.  
Move the blue player with the arrow keys to reach the green finish panel while avoiding black walls. Each round is timed, and your score increases when you reach the finish successfully. Hitting a wall resets your position and increases the round number.  

**Features**  
- Arrow key movement  
- Collision detection with walls  
- Round timer (30 seconds)  
- Score and round tracking  
- Visual feedback when hitting walls  

**How to Play**  
1. Start the game.  
2. Use the arrow keys to move the blue panel.  
3. Reach the green panel before time runs out.  
4. Avoid touching the black walls.  

**Requirements**  
- Windows OS  
- Visual Studio or any C# compiler supporting Windows Forms
